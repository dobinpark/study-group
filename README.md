# 함공
함께 공부하는 공간, 함공입니다.

# 💡 Topic
스터디 그룹 관리 및 협업을 위한 서비스 플랫폼 개발을 통해 사용자들이<br>
효율적으로 학습 그룹을 형성하고 관리할 수 있는 서비스 제공

# 📝 Summary
함공은 '함께 공부하는'의 줄임말로, 학습자들이 지식을 공유하고 함께 성장할 수 있는<br>
온라인 스터디 커뮤니티 플랫폼입니다.<br>

사용자들에게 원활한 학습 경험을 제공하며,<br>
다양한 학습 자료와 정보를 한 곳에서 확인하고 공유할 수 있습니다.

# ⭐️ Key Function

<h3>스터디 그룹 관리</h3>
관심 주제별 스터디 그룹을 생성하고 관리할 수 있으며, 그룹 내 일정 조율과 진도 관리를 돕습니다.

<h3>알림 시스템</h3>
쪽지나 참여한 스터디 그룹 방장이 공지사항을 올릴 때 알림으로 표시해줍니다.

<h3>커뮤니티/고객센터 게시판 활성</h3>
사용자들이 서로 소통하며 함공이 보다 활성화될 수 있도록 구성하였습니다.
